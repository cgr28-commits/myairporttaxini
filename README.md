# My Airport Taxi NI Website

This is a GitHub Pages-ready static website for My Airport Taxi NI.

## Files

- `index.html` — main website page
- `style.css` — design and mobile layout
- `script.js` — mobile menu and quote calculator

## How to upload to GitHub

1. Open your GitHub repository: `cgr28-commits/Belfast-taxi-`
2. Delete the old ZIP file if it is still there.
3. Upload these files directly into the main/root area:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
4. Go to repository `Settings`
5. Go to `Pages`
6. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
7. Save.

Your site should then publish through GitHub Pages.
